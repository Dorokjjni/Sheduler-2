# PWA с шестью страницами

В приложении есть выдвижное меню, похожее по принципу навигации на Vernatun: кнопка с тремя полосками открывает панель, а выбор пункта показывает отдельный экран.

## Шесть страниц

1. Главная
2. Календарь
3. Регулярное
4. Планирование
5. Задачи и цели
6. Настройки

## Запуск на Windows

Откройте терминал в папке проекта и выполните:

```bat
py -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## Публикация на GitHub Pages

Загрузите все файлы и папку `icons` в корень репозитория. Затем включите GitHub Pages из ветки `main`, папки `/root`.

## Навигация

Адрес каждой страницы содержит хеш, например:

- `#home`
- `#calendar`
- `#regular`
- `#planning`
- `#tasks`
- `#settings`

Поэтому работают кнопки браузера «Назад» и «Вперёд».
